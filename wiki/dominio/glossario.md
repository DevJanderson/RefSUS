# Glossário

Siglas e termos que aparecem na doc e no código.

| Termo | Significado |
|-------|-------------|
| **CID-10** | Classificação Internacional de Doenças, 10ª revisão |
| **SUS** | Sistema Único de Saúde |
| **SINAN** | Sistema de Informação de Agravos de Notificação |
| **SIM** | Sistema de Informação sobre Mortalidade |
| **SIH** | Sistema de Informações Hospitalares |
| **SINASC** | Sistema de Informações sobre Nascidos Vivos |
| **DATASUS** | Departamento de Informática do SUS |
| **MS** | Ministério da Saúde |
| **SVS / SVSA** | Secretaria de Vigilância em Saúde e Ambiente |
| **IBGE** | Instituto Brasileiro de Geografia e Estatística |
| **DOU** | Diário Oficial da União |
| **SE** | Semana Epidemiológica (1 a 52/53) |
| **SRAG** | Síndrome Respiratória Aguda Grave |
| **CIR** | Região de Saúde (Comissão Intergestores Regional) |
| **Agravo** | Doença ou condição de notificação compulsória |
| **Nowcasting** | Correção estatística do atraso de notificação |
| **Notificação imediata** | Até 24h (agravos graves ou de potencial epidêmico) |
| **Notificação semanal** | Até o fim da semana epidemiológica |
| **Ficha SINAN** | Formulário de notificação por agravo |
