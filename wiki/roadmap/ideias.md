# Ideias soltas

Caderno de rascunhos. Não é roadmap oficial — é estoque de ideias pra discutir antes de virar feature.

## Ideias

- **CLI `refsus`** — consultar dados direto do terminal (`refsus cid A90`)
- **SDK TypeScript** gerado do OpenAPI, publicado no npm
- **Webhooks** de atualização quando dataset muda
- **GraphQL** como camada alternativa a REST (só se surgir demanda real)
- **Dashboard embeddable** — `<iframe>` pra secretaria municipal colocar no site
- **Busca fuzzy** em doenças com Fuse.js ou FlexSearch
- **Feeds RSS** por agravo (quando expandir pra epidemiológico)
- **Integração FHIR** — mapear recursos do RefSUS pro padrão HL7 FHIR
- **Análise de tendência** por município (quando tiver dados SINAN)

## Como promover pra roadmap

Se uma ideia ganhou tração:
1. Criar doc dedicado em `roadmap/`
2. Remover daqui
3. Adicionar linha na tabela de status de `roadmap/README.md`
